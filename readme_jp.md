# Helix 5 rows JIS layout

このキーマップは、 [qmk_firmware に同梱されている `five_rows_jis`](https://github.com/qmk/qmk_firmware/tree/master/keyboards/helix/rev2/keymaps/five_rows_jis) を元に作成されました。


## コンセプト

このキーマップは日本語JISキーボード配列を利用しているユーザーが無理なく操作出来るというコンセプトで作成しました。以下の特徴があります。  

* 日本語の長音記号をレイヤーを移動せずに入力可能なように考慮しています
* 入りきらない記号関連を Raise レイヤー配置しています


### NORMALキーマップ

* Nキーを左人差し指で、Bキーを右人差し指で押下することが可能

## 配列

### NORMAL

　Adjust + ModNrmキーでNORMALキーマップに切り替わります。
